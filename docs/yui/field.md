# field

#### basic usage
<demo-field demo="1"></demo-field>
```vue
<yn-field-group>
  <yn-field v-model="basicUsage" placeholder="请输入内容"></yn-field>
</yn-field-group>
```

#### numeric type
<demo-field demo="2"></demo-field>
```vue
<yn-field
  v-model="phoneNumber"
  label="手机号码"
  type="number"
  min="3"
  max="10"
  placeholder="phone number"
  pattern="[0-9]*"
></yn-field>
```
